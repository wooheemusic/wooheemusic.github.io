Cover Letter

I have started to study and work on web applications from the beginning of last year. I thought I had to be a developer whom BIG entrepreneurs want to hire. I read a book 'How Google Works' then I found out the CEO of Google wants to hire developers who are creative and professional and study on their own. So I have been striving to become like them.

I had thought I could finish the development of the application very easily and quickly but it has took more than a year to make it to fit my very first initiative, a gmail-like single page web application with the newest features of open APIs in compliance with the fundamental concepts of software engineering(server architectures, normalized relations, web protocols and so on).

I did not intend to create a single page wep application(SPA) framework 'SectionFlow', but it was a necessary output from consideration of maintainability. It can enables html developers to develop a SPA with ONLY HTML and a little javascript configuration as if Spring framework do with POJOs and some configuration.

I did well enough even in my early and narrow sight. If I meet and work with people who are excellect, I will do more than I did last one and half year.
